What is Termux?

Termux is a powerful Linux terminal emulator for Android. It allows you to install and run Linux packages without root access, turning your phone into a portable cybersecurity lab.

For education & authorized testing only.

Installation (2026)

  1. Install F-Droid
  2. Search and install Termux
  3. Open Termux and allow bootstrap packages
